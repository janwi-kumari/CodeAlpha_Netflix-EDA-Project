# Netflix Content Analysis

## Project Overview

This project performs Exploratory Data Analysis on the Netflix Dataset containing 8,807 records.

## Tools Used
- Python
- Pandas
- Matplotlib
- VS Code

## Dataset Features
- Show ID
- Type
- Title
- Director
- Cast
- Country
- Release Year
- Rating
- Duration
- Listed In
- Description

## Key Findings
- Movies account for approximately 70% of Netflix content.
- The United States contributes the heighest number of titles.
- TV-MA is the most comman rating.
- 2018 recorded the heighest number of releases.
- Anupam Kher appears most frequently in the dataset.
