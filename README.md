# CodeAlpha_Netflix-EDA-Project
Exploratory Data Analysis(EDA) of Netflix titles dataset using Python, Pandas, Matplotlib to uncover content trends, ratings, genres, countries, and release patterns.

The project performs Exploratory Data Analysis (EDA) on the Netflix Titles dataset containing 8807 records. The objective is to identify content trends, popular genres, ratings distributions, country-wise contributions, and release patterns using Python, Pandas and Matplotlib.
